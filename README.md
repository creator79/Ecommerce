### Built With
This project is built with ReactJs, Redux,  React-Router-Dom, Axios, PrivateRoutes

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/creator79/Ecommerce.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run the project
    ```sh
    npm run dev
    ```
4. Open the project in browser
    ```sh
    http://localhost:5173/
    ```
